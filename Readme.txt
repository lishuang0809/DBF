This package is a collection of code used in the paper "Digital Beamforming Robust to Time-Varying Carrier Frequency Offset". The code in this package is sufficient to reproduce all of the results in this paper.

The code has been tested to run in MATLAB R2020a.

If you use this codes in an academic paper, please cite our paper:
S. Li, P. Nayeri, and M. B. Wakin, Digital Beamforming Robust to Time-Varying Carrier Frequency Offset, arXiv:2103.04948, 2021.
